- `textAnchor` set to `'start'` when `x > cx`, `'end'` otherwise
- `dominantBaseline="central"`
- `fontSize={12}`
- Text content: `${name}: ${value.toLocaleString()}`
- Truncate store names longer than 20 characters with ellipsis before the colon

**Tooltip (per D-08):**
Use `<Tooltip>` with a custom `formatter` that shows unit count and a `labelFormatter` or custom content that shows percentage. Set `contentStyle` to use CSS variables for dark mode compatibility:

```typescript
contentStyle={{

  fontSize: '13px',

}}
```

```
      </Pie>
      <Tooltip ... />
```
