### Integration Gaps Found

**INT-01 (Critical — MATCH-05, MATCH-06): rou_data schema has no is_ranged column**

- `match.ts` line 130 constructs `RouItem` with `isRanged: false` for all rows — hardcoded, not from data
- `matchTransfers` sort (`dest.isRanged ? -1 : 1`) never activates ranged-first grouping
- Functional impact: ranged items do not float to the top; sort is ROU-descending only
- Root cause: `rou_data` schema has no `is_ranged` column (noted in 04-01-SUMMARY decisions)
- `parseIsRanged()` and `dead_stock.is_ranged` storage are correct — gap is in the ROU upload pipeline
- Fix: add `is_ranged BOOLEAN DEFAULT false` to `rou_data` table + update upload.ts ROU INSERT
