Make two targeted changes to the table container elements:

**Change 1 — Outer table container** (currently `<div className="border border-[var(--color-border-light)] rounded-lg overflow-hidden">`):

Add an inline style attribute so the element becomes:

```tsx
<div
  className="border border-[var(--color-border-light)] rounded-lg overflow-hidden"
  style={{ overflowX: 'auto' }}
>
```

The `overflow-hidden` Tailwind class and the `overflowX: 'auto'` inline style coexist — the inline style takes precedence for the x-axis, while `overflow-hidden` still clips vertical overflow on the outer boundary.

**Change 2 — Virtualized body scroll container** (currently `style={{ overflowY: 'auto', maxHeight: 'calc(100vh - 300px)', position: 'relative' }}`):

Add `overflowX: 'auto'` to the existing inline style object:

```tsx
style={{ overflowY: 'auto', overflowX: 'auto', maxHeight: 'calc(100vh - 300px)', position: 'relative' }}
```

Do not remove any existing properties. Do not touch the sticky header div or grid divs in this task.
