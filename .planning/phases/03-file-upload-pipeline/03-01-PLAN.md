    **parseCSV(buffer: ArrayBuffer): string[][]**
    - Decode with `new TextDecoder('utf-8').decode(buffer)`
    - Strip BOM: `if (text.charCodeAt(0) === 0xFEFF) text = text.slice(1)`
    - Normalise line endings: `text.replace(/\r\n/g, '\n').replace(/\r/g, '\n')`
    - Split on `\n`, filter empty lines
    - Parse each line respecting quoted fields (commas inside quotes)
    - Return `string[][]`

    **parseXLSX(buffer: ArrayBuffer): string[][]**
    - `import * as XLSX from 'xlsx'`
    - `const wb = XLSX.read(buffer)`
    - `const ws = wb.Sheets[wb.SheetNames[0]]`
    - `return XLSX.utils.sheet_to_json<string[]>(ws, { header: 1, defval: '' })`
    - Convert all cell values to strings

    **findHeaderRow(rows: string[][], required: string[]): number**
    - Iterate rows 0..N. For each row, check that every required canonical name has at least one alias present in the row values (case-sensitive match on trimmed values).
    - Return index of first matching row, or -1.

    **buildColumnMap(headerRow: string[]): Record<string, number>**
    - For each cell in headerRow, look up in HEADER_ALIASES. If found, map canonical name to column index.
    - Return `{ 'Item Code': 0, 'ROU': 3, ... }`

    **parseRouFile(buffer: ArrayBuffer, filename: string): RouRow[]**
    - Call parseCSV or parseXLSX based on `filename.toLowerCase().endsWith('.xlsx')`
    - `findHeaderRow(rows, ['Item Code', 'ROU', 'SOH'])` — throw `Error('Required headers not found: Item Code, ROU, SOH')` if returns -1
    - `buildColumnMap(rows[headerIdx])`
    - Iterate rows after header. Skip rows where Item Code cell is empty/whitespace.
    - Extract: `sku = cell.trim()`, `description = descCell?.trim() ?? ''`, `rou = parseFloat(rouCell)` (produces NaN if invalid — do NOT default to 0), `soh = parseFloat(sohCell)` (same NaN rule)
    - Return `RouRow[]`

    **parseDeadStockFile(buffer: ArrayBuffer, filename: string): DeadStockRow[]**
    - Same CSV/XLSX dispatch
    - `findHeaderRow(rows, ['Item Code', 'SOH'])` — throw `Error('Required headers not found: Item Code, SOH')` if returns -1
    - Extract: `sku`, `description`, `soh = parseFloat(sohCell)`, `isRanged = RANGED_TRUTHY.has(rangedCell?.trim().toLowerCase() ?? '')` (if Ranged column missing, default false)
    - Unrecognised columns silently dropped (only mapped columns are read — per D-15)
    - Return `DeadStockRow[]`

    function csvToBuffer(text: string): ArrayBuffer {
      return new TextEncoder().encode(text).buffer;
    }
    function bomCsvToBuffer(text: string): ArrayBuffer {
      const bom = new Uint8Array([0xEF, 0xBB, 0xBF]);
      const content = new TextEncoder().encode(text);
      const combined = new Uint8Array(bom.length + content.length);
      combined.set(bom);

      return combined.buffer;
